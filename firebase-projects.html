Realtime Firebase Database - Custom API

Example of database structure from custome API made to post feedback to students

The endpoint shows how my data was structured. To explain some context, for our server-side scripting course (NodeJs), as our final project, we had to write an api. My api invovled feedback system that stores and tracks feedack given to students on writing assignments. The biggest challenge was organizing the data in away that was human-legible, worked in JSON, and allowed for my front-end to easily display the data. I also wanted everything online so I chose Firebase which is very trendy and well known database system.
